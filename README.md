# Calculator Website

## Project Description
This project is a web-based calculator application developed using HTML, CSS, and JavaScript. 
It is designed to perform all standard arithmetic operations commonly available in mobile phone calculators, 
including addition, subtraction, multiplication, division, decimal calculations, and continuous operations.

The application also maintains a history of previously performed calculations, allowing users to review past results.
The website is hosted using GitHub Pages and is implemented as a fully client-side application without the use of Git commands or backend technologies.

## Features
- Performs all basic and standard calculator operations
- Supports decimal and continuous calculations
- Displays history of previous calculations
- Clear display and clear history functionality
- Clean and professional user interface
- Fully client-side web application

## Technologies Used
- HTML for webpage structure
- CSS for styling and layout
- JavaScript for calculator logic and history management

## Usage
1. Enter numbers using the calculator interface.
2. Select the required arithmetic operation.
3. Perform continuous calculations as needed.
4. Press the equals button to view the result.
5. View previously executed calculations in the history section.
6. Use the clear options to reset the display or calculation history.

## Objective
The objective of this project is to understand and implement core frontend web development concepts, 
apply JavaScript for real-time calculations, and deploy a static web application using GitHub Pages.
